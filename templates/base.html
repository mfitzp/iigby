Data is sourced from the European Centre for Disease Control daily report data on the geographic distribution of COVID-19 cases worldwide . Updated daily at 00:00 UTC.

Daily cases and deaths for each country are smoothed using a 7-day window function. Change in the daily rate is calculated using a 7-day rolling difference. Values above zero indicate the daily number of cases or deaths is increasing, while those below zero indicate that the number of cases or deaths is decreasing.
